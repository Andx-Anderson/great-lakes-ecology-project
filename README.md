# Sustainable City Design for Wolf and Deer Conservation

A biology/ecology project focused on reducing habitat fragmentation and human-wildlife conflict in the Great Lakes region.

This project explores how wolf and deer populations are affected by human development and proposes a sustainable city design that supports both ecosystems and urban growth.

---

## Project Overview

- Region: Great Lakes (USA)
- Key Issue: Habitat fragmentation
- Species: Wolves (predator) & Deer (prey)
- Goal: Design a city that reduces ecological conflict while maintaining human development

---

## Key Features

- Data Visualization
  - Wolf population trends
  - Deer vs wolf density comparison

- Food Web Model
  - Shows ecosystem relationships (plants → deer → wolves → humans)

- Sustainable City Design
  - Wildlife corridors
  - Buffer zones
  - Renewable energy systems
  - Transportation planning

- Implementation Plan
  - 4-phase timeline (planning → infrastructure → development → monitoring)

- Policy Recommendations
  - Land use limits
  - Zoning for buffer zones
  - Wildlife protection laws

- Ecological Monitoring
  - Wolf tracking (GPS, camera traps)
  - Biodiversity indicators
  - Habitat connectivity metrics

- Risk Analysis
  - Cost, public resistance, environmental uncertainty
